Previous design

My previous website was starting to lose its charm and needed a design that captilized on HTML5/CSS3 trends.

MOCKUP

To experiment with layout and color schemes, I moved into Fireworks to render some mockups.

Iteration 1

Next step was implementing the mockups into HTML5/CSS3.

Iteration 2

I liked how Ryan Sim used a left side navigation, so I decided change the framework of my website and strip down the interface to a more minamlistic style.

Iteration 3

I utilized some CSS transitions to showcase UX interactions and played around with the load screen. See the results

Current

After gathering more user feedback, I reverted back to the original grid design. With my new understanding in CSS transitions, I enhanced the Iteration1 version with more dynamic images and loading screen.

NEXT STEPS

There is still a lot I want to do with CSS3 animations and transitions. Ideally, each image is interactive and showcases the design in its true form. Michael Farag's portfolio is an inspiring piece of work that shows a bit of what I am aspiring to do.
